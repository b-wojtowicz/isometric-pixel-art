About

Isometric pixel art is a plugin for GIMP that generates isometric graphics.

This tool was created for game developers to help them create temporary isometric assets during game production before final graphic is delivered by artists.

If you're using this different way - believe me it's also cool.

Installation notes

Just copy this script to your GIMP scripts folder.
